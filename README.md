# wordalopod
Wordalopod CLI  Wordalopod CLI est un jeu de devinette de mots développé en Python et joué directement dans le terminal. Le joueur doit retrouver un mot secret en un nombre limité de tentatives. Après chaque essai, le jeu fournit des indices visuels à l’aide de couleurs pour indiquer si les lettres sont bien placées, mal placées ou absentes du mot.
